      Root Cause Agreement — What is it?
      Of the 5 most similar historical incidents retrieved, how many share the same root cause as the predicted one? If 5/5 agree → very high confidence. If only 2/5 agree → the prediction is less certain and you should review manually.
      🎯
      Severity Agreement — What is it?
      How many of the 5 retrieved incidents share the same severity level? This is often lower than root cause agreement because severity is an administrative label — not described in the incident text itself. This is expected behaviour.
      📐
      Top Match Similarity — What is it?
      How semantically close is the single closest historical incident to your query? Converted from FAISS L2 distance to a 0–100% scale. Above 80% = very strong match. Below 50% = weak match, use with caution.
      📏
      Similarity Spread — What is it?
      The difference between the best and worst of the top-5 similarities. A small spread (below 10%) means all 5 matches are equally strong — tight cluster, reliable result. A large spread means only the top match is good and the others are weaker.

      About intelliCAPA

      intelliCAPA uses semantic AI to match pharmaceutical incidents against a corpus of historical CAPA records. Built on SentenceTransformers and FAISS, it surfaces actionable recommendations in under a second — no retraining required.

      Processing Pipeline
      1

      Incident Description

      Free-text description of the quality or compliance incident.

      2

      Sentence Transformer Embedding

      Encoded into a 384-dimensional vector using all-MiniLM-L6-v2.

      sentence-transformers384-dim
      3

      FAISS Semantic Retrieval

      Top-5 nearest neighbours retrieved by L2 distance from pre-built index.

      faiss-cpuk=5
      4

      Root Cause & Severity Voting

      Majority vote across top-5 matches determines predicted root cause and severity.

      5

      Historical CAPA Lookup

      Records filtered by matching root cause and severity to build the candidate pool.

      6

      Action Ranking

      Actions ranked by frequency — most consistently applied interventions first.

      7

      Explainability Output

      Full evidence trail — similar incidents, CAPA records, confidence, and reasoning.
